## 📁 Файлы персонажей

Каждый персонаж представлен в формате JSON со следующими полями:

```json
{
  "name": "Имя персонажа",
  "description": "Подробное описание роли, возраста, специализации и предпочтений",
  "personality": "Черты характера и стиль общения",
  "scenario": "Как персонаж взаимодействует с командой и выполняет задачи",
  "first_mes": "Первое сообщение персонажа при начале работы",
  "mes_example": "Пример диалога с другими членами команды",
  "tags": ["список", "тегов", "для", "поиска"]
}
```

## 🎯 Как использовать

1. **Загрузка персонажей**: Импортируйте JSON-файлы в вашу систему (например, SillyTavern, Character.AI или другую платформу)
2. **Настройка сценария**: Используйте поле `scenario` для определения контекста взаимодействия
3. **Запуск диалога**: Начните с `first_mes` для инициации работы команды
4. **Симуляция работы**: Используйте `mes_example` как шаблон для реалистичных диалогов
